# RandomNumber

**Random Number Generator**

This simple console application generates random numbers at regular intervals and prints them to the console. Below are instructions on how to compile and run the application.

## Installation

To use this application, follow these steps:

1. **Clone the repository:**

`git clone https://github.com/your_username/random-number-generator.git`

2. **Navigate to the project directory:**

`cd random-number-generator`

3. **Compile the program:**

`csc Program.cs`

## Usage

Once you've compiled the program, you can run it using the following steps:

1. **Execute the compiled executable:**

`Program.exe`


2. **Observe the output:**
The program will print random numbers between 1 and 100 to the console every 3 seconds.
